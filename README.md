# CreatorAI Studio 🎬

**AI-Powered YouTube Content & Monetization Planner**

CreatorAI Studio is a cutting-edge SaaS platform that empowers YouTube creators to generate viral content ideas, complete scripts, and monetization strategies using artificial intelligence.

## 🚀 Features

### Core Features
- 🎯 **Viral Content Generator** - AI-powered content ideas based on trends
- 📝 **Script Writing** - Complete YouTube Shorts and cartoon story scripts
- 📊 **Content Planning** - 30, 60, and 90-day content roadmaps
- 💰 **Monetization Strategy** - Detailed revenue roadmaps and income projections
- 🔍 **SEO Optimization** - Titles, tags, descriptions, and hashtags
- 📈 **Market Analysis** - Trend tracking and competition analysis
- 🎨 **AI Prompts** - Image and video generation prompts

### Premium Features
- ✅ Unlimited Content Ideas
- ✅ Unlimited Scripts
- ✅ Unlimited AI Prompts
- ✅ 90-Day Planner
- ✅ Daily Market Analysis
- ✅ AI Thumbnail Generator
- ✅ Competitor Analysis
- ✅ Revenue Prediction
- ✅ Personal AI Content Coach

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **State Management**: Zustand
- **UI Components**: Lucide Icons
- **Notifications**: React Hot Toast
- **Theme**: Next Themes (Light/Dark mode)

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/kedsonscreations-create/vishesh-rajput.git

# Navigate to project
cd vishesh-rajput

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit `http://localhost:3000` to see the application.

### Build for Production

```bash
npm run build
npm start
```
